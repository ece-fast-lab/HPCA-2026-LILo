# Step 4: Generate Figures

## How to run
`python3 ./generate_figures.py`

The script will output figures `fig11.png` and `fig12.png`.

## Description
The script will:

(1) Read previous collected data from `../scripts/step_1_baseline/results/`, `../scripts/step_2_decomp/results/`, and `../scripts/step_3_storage_offload/results/`.

(2) Process the data and generate figures.
